Online

The first webinar that I attended was a webinar about how JavaScript frameworks ensure fully integrated, secure and UI rich web systems. It was hosted by Softweb Solutions Inc. The talk was about JavaScript and trends in web development, web frameworks available in the market, and business challenges and benefits of implementing web frameworks, and followed up by a question and answer segment. It was very informative and the host was straightforward. It is a shame they didn’t have much time to answer questions. Although the webinar was mainly focused on business, I chose this webinar because I think that I will be able to apply what I learnt during my upcoming internship. Furthermore, some things were mentioned that I didn’t know was an industry standard, for example, the use of node.js. I am currently learning how to use that in my programming class and I was surprised to find the host mentioning it. The host also briefly showed us how to use it. I think I now have a better understanding of Node.js and JavaScript and this would give me an edge for my assignments and upcoming internship.

The second webinar that I attended isn’t a webinar, it’s an online boot camp about Git and Github sprint. It was hosted by Le Wagon Singapore. In this boot camp, they talked about the basics of code versioning and collaborations. The boot camp was very informative and engaging, the host, Guido, is a great teacher and he was eager to answer questions thrown at him. He was also patient with us. Because it is a boot camp, there was a live demo on how to set up Git and Github and Guido also showed us how to use it. I learnt a lot from this boot camp, it taught me how to better manage my projects especially for when I am working in a group. I also learnt how to host my repository and publish it online. I used this to host my portfolio so it is easily accessible. Furthermore, a few days after the boot camp, I was introduced to Github again during my programming class. At this point, I had everything set up and ready to go and have a better understanding of how Github works. I think this online boot camp built a strong base about Github for me, and I will be able to use all this knowledge to teach and lead my teammates in our upcoming assignments where we will be using Github to collaborate.
